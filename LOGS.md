# Просмотр логов Kira Mind Bot

## На сервере (Docker)

После деплоя на сервер (165.232.120.123) логи контейнера смотри так:

```bash
ssh <user>@165.232.120.123

# Последние 200 строк логов
docker logs source-kira-mind-bot-1 --tail 200

# Логи в реальном времени
docker logs source-kira-mind-bot-1 -f

# Поиск ошибок
docker logs source-kira-mind-bot-1 2>&1 | grep -i "ошибка\|error\|❌"
```

Имя контейнера может отличаться (например `enigma-bots-kira-mind-bot-1`). Список контейнеров:

```bash
docker ps -a | grep kira
```

## Ошибки сохранения в память

При сбое записи в векторную БД (Qdrant) или при вызове OpenAI embeddings в логах будет:

- `❌ Ошибка сохранения факта в векторную БД:` + полный stack trace
- Причины: недоступность Qdrant, 500 от API, нехватка места на диске, таймаут сети

Пользователю в чат уходит короткое сообщение; детали — в логах контейнера.
